## ğŸ“ How to Use in Your System

### Load Policies in Python

```python
import json
from pathlib import Path

# Load all policies
policies_dir = Path("datasets/policies_corpus/policies_parsed")
policies = {}

for policy_file in policies_dir.glob("*.json"):
    with open(policy_file) as f:
        policy = json.load(f)
        policy_id = policy["policy_id"]
        policies[policy_id] = policy

# Access specific policy
cornell = policies["CORNELL_AI_2025_v1"]

# Get all allowed uses
allowed_uses = cornell["allowed_uses"]

# Check disclosure requirements
for use in allowed_uses:
    if use["disclosure_required"]:
        print(f"{use['use_case']}: {use['disclosure_method']}")
```
